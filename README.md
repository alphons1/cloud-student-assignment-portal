# AWS Cloud Storage & Analytics Platform

A cloud-based student assignment management system built using AWS and Python.

## Features

* Upload files to Amazon S3
* View cloud-stored files
* Analytics dashboard
* SQLite upload tracking
* AWS IAM authentication
* Streamlit frontend

## Tech Stack

* Python
* Streamlit
* AWS S3
* AWS IAM
* SQLite
* Plotly

## Architecture

User → Streamlit Frontend → Python Backend → AWS S3 + SQLite

## Future Improvements

* User authentication
* AWS RDS integration
* File downloads
* EC2 deployment
* Multi-user support
